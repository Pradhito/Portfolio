Robot Pemadam Api

Fire fighting robot. This robot is equipped with an infrared sensor as a distance sensor to detect obstacles, a micro switch as a bumper or emergency brake and a button to run the robot, a photodiode sensor as a fire detector. for processing this robot using arduino uno. while for the output, this robot has a dc motor as a driving force, as well as an extinguisher. how this robot works, the robot will start by detecting the distance of obstacles and fire, when there is no fire then the robot will start tracing the wall (left or right wall) using an infrared proximity sensor. if a fire is detected, the robot will point to the fire and extinguish the fire using a dc motor as a fan.
