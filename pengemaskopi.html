Pengemas Kopi Otomatis

Automatic coffee filler. this tool uses ultrasonic sensors, keypad, and loadcell as input, arduino uno as processor, and servo motor as output. how this tool works first we enter the number of grams via the keypad, then we put the container for the coffee, if the ultrasonic sensor detects the container then the next, the coffee will be poured using a servo while the coffee is being poured, the loadcell sensor reads how many grams of coffee have been filled. When the desired weight is reached, the coffee stops being filled, and the container is pushed forward using a servo.
